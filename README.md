# 玛德琳角色设计

## 遗物

- [x] 登山背包 `Climbing Backpack` 初始遗物。在每场战斗开始时，将一张深呼吸放入你的手牌。
- [x] 不知所谓的机器 `Pointless Machines` 你每打出1张冲刺，获得2点格挡。在打出24张冲刺后升级。
- [x] 沉重与脆弱 `Heavy and Frail` 你每打出3张冲刺，获得12点格挡。
- [ ] 草莓派 `Strawberry Pie` 你每消耗6点能量，对所有敌人造成10点伤害。

## 事件

- [ ] 无冲 `No Dash`

## 姿态

- [x] 着陆 `Land`：保持冲刺机会不小于上限。你无法积累动量。
- [x] 凌空 `Soar`：你不能打出跳跃。

## 能力

- [x] 冲刺机会 `Dash Chance` (点)：你需要冲刺机会来打出冲刺。上限为1点。
- [x] 动量 `Momentum` (点)：你打出的跳跃和冲刺可以额外选择与点数相等的牌。
- [x] 分散与失落 `Scattered and Lost` (点)：同卡牌。

## 卡牌关键字

- [x] 移动 `Move` （可以）选择一张手牌耗能打出，若无法打出则将其丢弃。
- [x] 跳跃 `Jump` 进入凌空。（移动。）获得1点动量。
- [x] 冲刺 `Dash` 消耗1点冲刺机会。（移动。）获得1点动量。
- [x] 充能 `Refill` 获得1点冲刺机会。
- [x] 永恒 `Eternal` 这张牌永远在你的手牌中。

### 技巧

冲刺

- [x] **横冲** `Dash Forward` 初始 ①
  > 冲刺。造成【9/12】点伤害。
- [x] **斜下冲** `Diagonal-down Dash` 常见 ⓪
  > 冲刺。造成【6/9】点伤害。
- [x] **切版冲刺** `Transition Dash` 常见 ①
  > 冲刺。造成【10/14】点伤害，结束你的回合。在你的下回合开始时充能。消耗。
- [x] **凯文** `Kevin` 罕见 ②
  > 冲刺。造成【15/19】点伤害，获得【1/2】点动量。
- [ ] **凌波微步** `Wave Dash` 罕见 ②
  > 冲刺。造成【12/15】点伤害。若这张牌被跳跃打出，获得2点能量。

跳跃

- [x] 原地跳 `Jump in Place` 初始 ⓪
  > 跳跃。获得【1/2】点临时敏捷，给予1层虚弱。
- [x] 小跳 `Hop` 常见 ⓪
  > 跳跃。获得【5/8】点格挡。
- [x] 兔子跳 `Bunnyhop` 常见 ①
  > 跳跃。获得【5/8】点格挡，获得【1/2】点动量。
- [ ] 中性跳 `Neutral Jump` 罕见 ①
  > 跳跃。抽一张牌。将这张牌放回你的手牌。
- [ ] **蹬墙跳** `Wallbounce` 罕见 ①
  > 跳跃。获得【8/10】点格挡。若你的动量不为0，造成【8/10】点伤害。

其他

- [ ] 速降 `Fastfall` 罕见 ①
  > 获得【2/3】点动量，你打出下一张牌后着陆。
- [x] *攀爬* `Climb` 罕见 ②
  > 获得等同于你当前动量【+1】的力量。在每回合结束时失去所有动量。
- [ ] *狼跳* `Coyote Time` 稀有 ①
  > 你每回合第一张跳跃可以在凌空时打出。

- [x] **刀片机关** `Blade` 罕见 ①

- [x] *提奥水晶* `Theo Crystal` 稀有 ③
  > 若你在本场战斗中【未受到攻击伤害/受到不多于一次攻击伤害】，战斗结束时额外获得一件遗物。
- [ ] *平静与坠落* `Quiet and Falling` 稀有 ②

### 7. 山顶

- [x] **检查点** `Checkpoint` 稀有 ①
  > 造成1点伤害。你在本场战斗中每获得过1点动量，这张牌的伤害增加【2/3】。
- [ ] 钥匙 `Key` 稀有 ②
  > 将一张【锁/锁+】放入你的【弃牌堆/抽牌堆】中。消耗。

    > 消耗所有手牌。每消耗一张牌，将一张【费用不为0的】随机攻击牌放入你的手牌，它们在本场战斗中耗能变为0。消耗。
- [ ] *粉色朝霞* `Pink Sunrise` 稀有 ①
