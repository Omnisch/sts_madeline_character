# 玛德琳角色设计

## 遗物

- [ ] 登山背包 `Climbing Backpack`：初始遗物。在你的回合结束时，保留所有的跳跃和冲刺。
- [ ] 不知所谓的机器 `Pointless Machines`

## 姿态

- [ ] 着陆 `Land`：保持不小于1点冲刺机会。失去所有动量。
- [ ] 落脚 `Stay`：保持不小于1点冲刺机会。你打出下一张牌时退出这一姿态。
- [ ] 攀附 `Grab`：你每打出一张牌，失去1点敏捷，着陆或落脚时恢复。
- [ ] 凌空 `Soar`：在退出这一姿态前，你不能打出跳跃。

## 能力

- [x] 冲刺机会 `Dash Chance` (点)：你需要冲刺机会来打出冲刺。
- [ ] 动量 `Momentum` (层)：你获得与层数相等的力量。移动须额外选择与层数相等的牌。

## 卡牌关键字

- [ ] 移动 `Move`：获得1点动量。选择一张手牌耗能打出，若无法打出则丢弃。
- [ ] 跳跃 `Jump`：进入凌空。
- [x] 冲刺 `Dash`：只能在有冲刺机会时打出，失去1点冲刺机会。
